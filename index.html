Apredendo git e git hub

  • oque é o git : é um sistema de controle de versão de código, permite registrar mudanças no códigos e manter eles organizados e facilmente indentificaveis, ele mantem um histórico completo de todas as auterações que foram feitas no codigo.
  • git hub é: a rede social com os projetos enviado via git.
  • comitar é : gerar uma nova versão do seu projeto.
  • branch é : criar uma ramificação uma nova linha da atual versão do código para fazer modificações sem alterar a versão atual, após as alterações feitas na branch se (quiser) fazer marge para salvar as modificações.

Comandos do git

  • git init: é para inicializar o git
  • git status: é para ver como esta o status do nosso git
  • git add: é preparar arquivo para ser commitado dizer qual arquivo vai ser gerenciado exemplo: index.html, ou seja o arquivo foi add ao git
  • git commit -m "escrever alguma coisa": Cria uma nova versão do projeto
  • git log : Ver quais versões temos no nosso git
